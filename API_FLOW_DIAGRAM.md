## Performance Characteristics

- **Best Case**: 1 API call (first key works)
- **Worst Case**: 5 API calls + error (all keys fail)
- **Average Case**: 1-2 API calls (first or second key works)
- **Timeout per attempt**: 30 seconds
- **Maximum total time**: 150 seconds (5 keys Ã— 30s)

## Capacity Increase

With 5 API keys, you get:
- **5x quota capacity** (if each key has same quota)
- **5x redundancy** (system stays up even if 4 keys fail)
- **Better distribution** (load spread across multiple keys)
